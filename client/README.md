# client-frontend

## Project prerequest
```
yarn global add @vue/cli
yarn add vue-router@4
npm install axios
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
